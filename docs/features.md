# Features

## Current observable features

### Authentication
- login using DNI/CUIT
- remember entered DNI locally

### Customer dashboard
- view customer identity data
- view status
- view latest payment
- view current balance

### Payment information
- alias
- payment code
- barcode
- payment helper dialog

### Billing visibility
- invoices list
- receipts list

### Multi-customer support
- select current customer when multiple customers are associated to the same login

### Local utilities
- clipboard copy actions
- file download/save abstractions
- loading and error popups
